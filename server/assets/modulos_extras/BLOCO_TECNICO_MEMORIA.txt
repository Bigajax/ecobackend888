# BLOCO TÉCNICO DE MEMÓRIA ECO

## Quando Gerar
Apenas se intensidade ≥ 7

## Formato
Ao final da resposta, adicionar UM JSON puro (sem crases, sem comentários)

## Estrutura (usando campos do seu banco)
```json
{
  "emocao_principal": "string minúscula",
  "intensidade": "number 0-10",
  "tags": ["máx 4 itens", "específicas", "sem PII"],
  "dominio_vida": "área principal afetada",
  "padrao_comportamental": "o que disparou + padrão observado",
  "nivel_abertura": "1-3",
  "categoria": "emocional|relacional|existencial|profissional|corpo|outro",
  "analise_resumo": "máx 20 palavras, foco no essencial"
}
```

## Melhorias na Forma de Usar

**1. PADRAO_COMPORTAMENTAL melhorado:**
- Era: descrição vaga do comportamento
- Agora: "gatilho + padrão" (ex: "sobrecarga_gera_perfeccionismo_defensivo")

**2. ANALISE_RESUMO mais direto:**
- Reduzido para 20 palavras (vs 25)
- Foco no insight essencial + necessidade da pessoa

**3. TAGS mais estratégicas:**
- Máx 4 itens (vs 5)
- Mais específicas para facilitar busca/agrupamento

## Regras Importantes
- analise_resumo: sem diagnósticos clínicos
- dominio_vida: trabalho, família, relacionamento, saúde, propósito, etc.
- tags: curtas, sem nomes/lugares específicos
- Nunca inventar detalhes sensíveis
