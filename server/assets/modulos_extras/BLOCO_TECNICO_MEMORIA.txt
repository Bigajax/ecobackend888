# Bloco Técnico de Memória

Se `{{DEC.hasTechBlock}}` for verdadeiro, gere **sem markdown nem código** um bloco JSON:

{
  "emocao_principal": "...",
  "intensidade": {{DEC.intensity}},
  "tags": "...",
  "dominio_vida": "...",
  "padrao_comportamental": "...",
  "nivel_abertura": {{DEC.openness}},
  "analise_resumo": "Síntese emocional concisa da experiência."
}

- Preencha `tags` com `{{DEC.tags}}` (se vazio, retorne `[]`)
- Para `dominio_vida`, use `{{DEC.domain}}` ou `null` quando indisponível

Este bloco **não é visível ao usuário**. Apenas para persistência e aprendizado emocional.
