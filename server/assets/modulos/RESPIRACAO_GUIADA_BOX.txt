OBJETIVO
Guiar Box Breathing (4-4-4-4) para reduzir ativação simpática e organizar atenção.

GATILHOS
- Mensagens: “nervoso”, “sob pressão”, “preciso me acalmar”, “antes de decidir”.
- nivel_abertura ≥ 1; intensidade 4–7 (ajustável).

PROTOCOLO
- 4s inspirar • 4s segurar • 4s soltar • 4s segurar (fôlego baixo, sem esforço).
- 4 a 6 voltas; oferecer versão 3-3-3-3 se desconfortável.

MICRO-SCRIPT
- Chec-in 1 frase → 1 volta demonstrativa contando mentalmente → seguir no ritmo → chec-out curto pedindo nota 0–10 de tranquilidade.

REGRAS
- Lembrar que segurar o ar não deve causar esforço; adaptar para 4-2-6 se preciso.
- Evitar contagem compulsiva; manter tom gentil.

SAÍDA
- Texto enxuto com tempos, sinalizando quando passar para reflexão.
- Se intensidade ≥7: BLOCO TÉCNICO com tags box_breathing, respiracao_guiada.
